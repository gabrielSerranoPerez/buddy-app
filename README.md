## Overview
Buddy App is a web-based assistive application designed to improve accessibility and digital inclusion for elderly users. The application focuses on simplified touch-based interaction and multi-device access through a web browser.

The goal of Buddy App is to provide elderly users with easy access to communication, media, and personal information while allowing relatives to stay informed.

## Key Features
- **Communication Access**
  - Touch-friendly interface for initiating phone calls or contact with trusted individuals

- **Media & Entertainment**
  - Integrated media links for news channels and entertainment content
  - Simple navigation designed for reduced complexity

- **Personal Information Dashboard**
  - Displays essential user information accessible to authorized relatives
  - Supports sharing important personal details for safety and assistance
